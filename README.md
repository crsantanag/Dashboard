# DashboardChar

Indicadores con HTML, CSS y JavaScript; Chart.js y API

## Descripción

Este proyecto permite graficar una serie de indicadores económicos que se usan en Chile (Imacec, Tpm, Uf, Ipc, Dólar y euro) entre los años 2000 y 2023.

## Inicio

Estas instrucciones te ayudarán a obtener una copia del proyecto y a ejecutarlo en tu máquina local para fines de desarrollo y prueba.

### Pre-requisitos

No se requieren instalaciones adicionales para ejecutar este proyecto, solo necesitarás un navegador web actualizado.

### Instalación

Sigue estos pasos para ejecutar el proyecto localmente:

1. Clona este repositorio en tu máquina local.

```bash
git clone https://github.com/crsantanag/Dashboard.git
```

2. Navega hasta el directorio del proyecto.

```bash
cd Dashboard
```

3. Abre el archivo `index.html` en tu navegador web.

## Ejecutando las pruebas

Este proyecto no cuenta con pruebas automatizadas. Sin embargo, puedes probar manualmente todas las funcionalidades del graficador de indicadores económicos.

### Analizar pruebas de extremo a extremo

Puedes verificar que los registros se cargen desde la API (mindicator.cl) y se grafiquen.

### Pruebas de estilo de codificación 

Asegúrate de mantener un código limpio y bien estructurado siguiendo las mejores prácticas de HTML, CSS y JavaScript.

## Implementación

Puedes implementar este proyecto en un servidor web o utilizarlo localmente sin necesidad de implementación adicional.

## Construido con

Herramientas utilizadas para crear el proyecto:

* HTML - Estructura del sitio web.
* CSS - Estilos y diseño visual.
* JavaScript - Lógica y funcionalidad del graficador.  Uso de funciones síncronas y asíncronas en JavaScript. Acceso a API.
* Chart.js - Biblioteca que permite graficar.
